# ERC-8004 Agent Creator

> On-chain AI agent identity registration — ERC-8004 standard implementation.

## Overview

Single-page dApp for registering AI agents as NFTs on EVM blockchains. Includes 5 multi-language SDKs, search service, CLI, and VS Code extension.

## Structure

- `index.html` — Main dApp
- `sdks/` — Python, Go, Rust, Java, Swift
- `search-service/` — TypeScript search & indexing
- `cli/` — CLI tool
- `mcp-server/` — MCP server
- `locales/` — 20 languages
- `docs/` — Docs
